# UI Agent

UI‑Agent — это специализированный агент CTOR‑H‑AI, отвечающий за анализ и улучшение пользовательского интерфейса платформы.  
Он работает только с визуальным слоем (HTML, CSS, JS) и не затрагивает игровую логику или правила CTOR.

## Основные задачи

- Анализировать текущий UI и выявлять слабые места.
- Предлагать улучшения интерфейса и UX.
- Оптимизировать структуру HTML/CSS.
- Улучшать читаемость и визуальную иерархию.
- Повышать доступность (контраст, размеры, кликабельность).
- Предлагать адаптацию под мобильные устройства.
- Генерировать рекомендации по рефакторингу UI‑кода.
- Улучшать визуализацию игровых действий (подсветка, анимации, состояния).

## Ограничения

- Агент не изменяет игровую логику.
- Агент не работает со стратегией ИИ.
- Агент не модифицирует правила CTOR.

## Структура папки

- `spec.md` — спецификация агента.
- `tasks/` — задачи, поставленные агенту.
- `suggestions/` — предложения и улучшения, сгенерированные агентом.
- `analysis/` — отчёты об анализе UI.

## Статус

Агент находится в стадии проектирования.  
Функциональность будет расширяться по мере развития платформы CTOR‑H‑AI.
