# Task: Analyze UI Version v1

## Description
Проанализировать пользовательский интерфейс версии **v1** из `version-map.json` и подготовить подробный отчёт о качестве UI, UX, адаптивности и визуальной структуре.

## Target Version
v1  
URL: https://a-ivs-h-vercel-j2ggfop68-bva5-7066s-projects.vercel.app/

## Objectives

1. **Анализ DOM‑структуры**
   - выявить избыточные или вложенные элементы
   - проверить логическую иерархию
   - определить проблемные места

2. **Анализ CSS**
   - найти дублирующиеся стили
   - выявить избыточные правила
   - оценить читаемость и структуру

3. **Анализ UX**
   - оценить понятность интерфейса
   - проверить визуальные подсказки
   - оценить удобство взаимодействия

4. **Адаптивность**
   - проверить отображение на мобильных устройствах
   - выявить проблемы с масштабированием
   - предложить улучшения

5. **Доступность**
   - проверить контрастность
   - размеры кликабельных элементов
   - читаемость текста

## Expected Output

UI‑Agent должен создать отчёт в:
agents/ui-agent/analysis/v1-report.md


Отчёт должен содержать:

- список найденных проблем
- рекомендации по улучшению
- примеры исправленного кода (если применимо)
- общую оценку UI‑версии
