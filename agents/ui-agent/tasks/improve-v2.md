# Task: Improve UI Version v2

## Description
Проанализировать текущую основную UI‑версию **v2** и подготовить улучшенную версию интерфейса.  
Цель — сохранить сильные стороны v2 (чистота, минимализм, структура), но устранить выявленные недостатки и повысить качество UI/UX, адаптивности и доступности.

## Target Version
v2  
URL: https://luminous-marzipan-e6de40.netlify.app/

## Objectives

### 1. Улучшение визуальной структуры
- усилить визуальную иерархию
- улучшить выравнивание элементов
- оптимизировать отступы и сетку
- повысить читаемость интерфейса
- **добавить анимацию исчезающих (съедаемых) фишек в стиле Shrink‑pop**
  - фишка быстро уменьшается (scale 1 → 0.8 → 0)
  - лёгкое уменьшение прозрачности (opacity 1 → 0)
  - длительность 150–220ms
  - анимация должна быть лёгкой и не мешать игровому процессу
  - должна работать на мобильных устройствах
  - реализовать через CSS (или CSS+JS для триггера)

### 2. Улучшение UX
- улучшить обратную связь элементов (hover, active)
- сделать кнопки более заметными и удобными
- улучшить логику расположения элементов
- повысить интуитивность интерфейса

### 3. Улучшение адаптивности
- проверить отображение на мобильных устройствах
- устранить проблемы с переполнением
- улучшить масштабирование элементов
- обеспечить корректное поведение сетки на разных экранах

### 4. Улучшение доступности (A11y)
- повысить контрастность текста и элементов
- увеличить кликабельные зоны
- улучшить читаемость шрифтов
- добавить aria‑атрибуты, если необходимо

### 5. Оптимизация DOM и CSS
- упростить структуру DOM
- удалить лишние вложенности
- оптимизировать CSS‑правила
- устранить дублирующиеся стили
