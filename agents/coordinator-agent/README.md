# Coordinator Agent

Coordinator Agent — центральный управляющий агент системы CTOR‑H‑AI.  
Он отвечает за распределение задач между агентами, выбор версий из `version-map.json`, запуск анализа, сбор результатов и координацию работы всей многоагентной системы.

## Основные функции

- Чтение карты версий (`version-map.json`)
- Назначение задач другим агентам
- Запуск UI‑Agent на разных UI‑версиях
- Сбор результатов анализа
- Ведение логов
- Управление очередью задач
- Поддержка будущих агентов (Game Logic, AI‑Player и др.)

## Статус

Агент находится в стадии проектирования.
