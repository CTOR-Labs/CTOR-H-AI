# Coordinator Agent — Specification

## 1. Назначение

Coordinator Agent — это управляющий агент, который координирует работу всех агентов в CTOR‑H‑AI.  
Он обеспечивает доступ к версиям, распределяет задачи, собирает результаты и управляет процессами анализа.

---

## 2. Основные функции

### 2.1. Управление версиями
- Читает `agents/version-map.json`
- Определяет доступные UI‑версии
- Определяет доступные engine‑версии (в будущем)
- Передаёт нужные ссылки агентам

### 2.2. Управление задачами
- Создаёт задачи для агентов
- Назначает задачи конкретным агентам
- Следит за выполнением
- Сохраняет результаты в `logs/`

### 2.3. Запуск UI‑Agent
- Передаёт UI‑Agent ссылку на UI‑версию
- Получает отчёты и рекомендации
- Сохраняет результаты в `ui-agent/suggestions/`

### 2.4. Масштабирование
- Поддержка нескольких агентов
- Поддержка нескольких версий
- Поддержка параллельных задач

---

## 3. Входные данные

- `agents/version-map.json`
- задачи из `tasks/`
- результаты агентов

---

## 4. Выходные данные

- логи выполнения
- отчёты об анализе
- распределённые задачи
- обновлённые статусы агентов

---

## 5. Ограничения

- Coordinator Agent не анализирует UI сам
- не изменяет код
- не принимает стратегические решения за разработчика
- не вмешивается в игровую логику
